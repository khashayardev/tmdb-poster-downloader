# 🎬 TMDB Poster Downloader

یک ابزار خودکار و سبک برای دانلود تمام پوسترهای فیلم‌های یک سال خاص از پایگاه داده TMDB با استفاده از GitHub Actions.

## 🤔 این ابزار چه کاری انجام می‌دهد؟

شما یک سال میلادی را وارد می‌کنید، و ابزار به صورت خودکار:

1. لیست تمام فیلم‌های آن سال را از TMDB دریافت می‌کند
2. شناسه IMDB هر فیلم را استخراج می‌نماید
3. پوستر اصلی هر فیلم را دانلود کرده و با نام `tt1234567.jpg` ذخیره می‌کند
4. همه تصاویر را در یک فایل ZIP قرار می‌دهد تا بتوانید دانلود کنید

## 🎯 برای چه کسانی مفید است؟

- **وبمسترها**: برای ساخت آرشیو لوکال تصاویر فیلم‌ها در سایت خود
- **توسعه‌دهندگان**: برای پروژه‌های دیتابیس فیلم و سریال
- **علاقه‌مندان به سینما**: برای جمع‌آوری آرشیو شخصی پوستر فیلم‌ها

## 🚀 نحوه استفاده

1. این رپازیتوری را **Fork** کنید
2. توکن API خود را از [TMDB](https://www.themoviedb.org/settings/api) بگیرید
3. در رپوی Fork شده، به **Settings → Secrets and variables → Actions** بروید
4. یک Secret جدید با نام `TMDB_ACCESS_TOKEN` بسازید و توکن خود را در آن قرار دهید
5. به تب **Actions** بروید و روی **Run workflow** کلیک کنید
6. سال مورد نظر را وارد کنید (مثلاً 2025) و بزنید **Run workflow**
7. چند دقیقه بعد، فایل ZIP را از بخش Artifacts دانلود کنید

## 📁 خروجی نهایی
posters_2025.zip >> 2025 >> tt0111161.jpg

نام هر فایل، شناسه IMDB همان فیلم است که کار تطابق با دیتابیس‌های دیگر را آسان می‌کند.


## ⚡ نیازمندی‌ها

- یک حساب رایگان در [TMDB](https://www.themoviedb.org)
- یک توکن Access Token (راهنما در بخش SECRETS_SETUP)
